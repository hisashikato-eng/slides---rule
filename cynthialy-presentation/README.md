# Cynthialy Presentation Skill

Cynthialy株式会社のブランドデザインに基づいたPowerPointプレゼンテーションを作成するためのClaudeスキルです。

## 特徴

- **統一されたブランドカラー**: ダークグリーン(#0D2623)とクリームイエロー(#E8DE9F)を基調
- **15種類のパターン**: スライドパターン8種 + チャートパターン7種
- **pptxgenjs使用**: Node.jsで実行可能

## 使用方法

1. このスキルをClaudeに読み込ませる
2. プレゼンテーションの内容を指示
3. 適切なスライドパターン/チャートパターンを使用してpptxgenjsコードを生成

## スライドパターン（パターン1〜8）

1. **タイトルスライド** - ダークグリーン背景、アクセントライン付き
2. **統計ボックススライド** - 3つの数値データを並列表示
3. **番号付きリスト** - 5項目程度の構造化リスト
4. **2カラム比較** - Before/After、従来/新規の比較
5. **セクション区切り** - 章の切り替え
6. **ステートメント** - 重要メッセージの強調
7. **3カラム支援内容** - ステップや支援内容の説明
8. **まとめ** - 3つのポイント + フッターメッセージ

## チャートパターン（パターン9〜15）

9. **縦棒グラフ** - 時系列データ、カテゴリ比較
10. **積み上げ棒グラフ** - 構成比の変化、セグメント内訳
11. **横棒グラフ** - ランキング、スコア比較
12. **ウォーターフォール** - 増減分析、ブリッジ分析
13. **バブルチャート** - 2軸ポジショニング、競争マップ
14. **折れ線グラフ** - トレンド推移、時系列比較
15. **スコアカード** - プログレスバー付き多次元比較

## インストール

Claudeのユーザースキルとして登録してください。
